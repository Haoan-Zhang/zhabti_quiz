# zhabti-northwesternu

这是第二版问卷网页，基于你提供的 10 道题和 24 种人格标签制作。

## 已包含内容

- 10 道单选题
- 24 种趣味人格结果
- 自动匹配逻辑
- 可直接部署到 GitHub + Vercel

## 部署方法

上传整个 `zhabti-northwesternu` 文件夹到 GitHub 仓库后，在 Vercel 导入仓库。

如果项目文件位于仓库子目录，请将 Root Directory 设为：

```text
zhabti-northwesternu
```

Framework Preset 请选择：

```text
Next.js
```

Build / Output / Install Command 全部保持默认，不要手动填写。
